Project overview

This is the the final project in the course Social Data Analysis and Visualization, taught at DTU in the spring of 2021.
It's about fires in London from 2009 to today. Some more text or something.
To get a quick overview of the project, please check out this introduction video below.
Links to an explainer notebook and a Github repository can be found at the bottom of the page.

Datasets

Below, each dataset used in this project is described.
Links to the datasets are also provided.

  • London Fire Brigade Incident Records

    This dataset contains the details of every incident responded by London Fire Brigade. It contains information about when and where the incident happened and the type of incident dealt with. The records are from January of 2009 to February of 2021. The dataset was obtained from here.

  • London Fire Brigade Mobilisation Records

    This dataset contains the details of every fire engine sent to an incident's scene. It contains information about the appliance used, where it was deployed from and response times. The records are from January of 2009 to February of 2021. The dataset was obtained from here.

  • Housing in London

    This dataset contains information about houses in London, all the way from January of 1995 to January of 2021. Average price by area, houses sold and population size are among the details provided. The dataset was obtained from here.

  • House ages in London

    This dataset contains the building periods of houses in London. The dataset was obtained from here.

  • London Borough Profiles

    This dataset paints a general picture of the boroughs of London by presenting a range of topics covering demographic, economic, social and environmental statistics for each borough. The data ranges from 2011 to 2017. The dataset was obtained from here.

  • UK postcodes

    This dataset contains all levels of postcodes in the United Kingdom. It is used to make a connection between other datasets which don't use the same postcode information in London. The dataset was obtained from here.
